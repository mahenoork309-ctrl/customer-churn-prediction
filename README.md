# Customer Churn Prediction — Random Forest

Predicting whether a customer will churn using Random Forest Classifier.

## Dataset
- Source: customer_churn_dataset.xlsx
- Features: Age, Tenure, Monthly Spend, Complaints, Satisfaction Score, etc.

## Tech Stack
Python | Pandas | Scikit-learn | Matplotlib | Seaborn

## Highlights
- Exploratory Data Analysis with 7 visualizations
- Feature Engineering (4 new derived features)
- Random Forest with class_weight balanced
- Evaluation: Accuracy, Precision, Recall, F1, ROC-AUC
- Hyperparameter Tuning via GridSearchCV
- Output with Churn Probability & Risk Labels
